Creates a ForeFlight Track Log CSV
To import:
  1. Send the file to your device (AirDrop, email, Files)
  2. ForeFlight → More → Track Logs → Import → From G1000 (CSV File)

Note: Track Log CSVs are generated as synthetic Garmin G1000 logs. ForeFlight will display the imported track as G1000; this is an app limitation and cannot be changed in the file. After importing, you can edit the log in ForeFlight to add or correct the aircraft tail number.
